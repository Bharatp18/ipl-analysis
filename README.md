\# IPL Match Analysis Dashboard



An end-to-end data analysis project analyzing 13 years of IPL data (2008-2022) to uncover match-winning patterns and build an interactive dashboard.



\## Key Insights

\- Winning the toss only improves win probability by \~51%

\- MI and CSK dominate win counts across all seasons

\- Fielding first has become more popular in recent seasons

\- Wankhede and Eden Gardens host the most high-pressure matches



\## Tech Stack

\- Python, Pandas, NumPy

\- SQL (SQLite)

\- Scikit-learn (Random Forest, Logistic Regression)

\- Plotly Dash

\- Jupyter Notebook



\## Project Structure

\- data/ - Raw datasets

\- notebooks/ - EDA and analysis notebooks

\- dashboard/ - Interactive Plotly Dash app

\- requirements.txt - All dependencies



\## How to Run

pip install -r requirements.txt

cd dashboard

python app.py



Open browser at http://127.0.0.1:8050



\## Dataset

IPL Complete Dataset from Kaggle (2008-2022)
